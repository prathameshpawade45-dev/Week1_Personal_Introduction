# Personal Introduction Program

## Description
This Python program takes the user's name, age, and hobby as input and displays a personal introduction message.

## Features
- Takes user input
- Displays formatted introduction
- Beginner-friendly Python project

## How to Run

1. Open terminal
2. Run:

```bash
python personal_intro.py
```

## Sample Output

Hello, Prathamesh
Your age is 21
Your hobby is Editing
Welcome to Python Programming!
